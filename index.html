WEB TYPOGRAPHIC POSTER

Inspiration

I wanted to create a minimalistic poster that would reflect my typeface, so I incorporated simple geometric motifs such as lines and circles to use within my poster. I looked at posters with simple typography as well, and looked at the way tehy aligned text and utilized grids.

Process

I began by experimenting with the layout of the poster, and how each element would relate to eachother. I also considered the colour palette I was going to use in this stage, and played around with how the viewer would use the page. I used position: fixed, which would add an interesting interactive element to the poster.

I played around with how layering the elements would look, but I ultimately wanted the poster to be readable, so I only made one of the larger typographic elements fixed.

In establishing the hierarchy of each part, I used a combination of size and colour. The biggest text would be highest in the heirarchy, and the body text was a colour more similar to the background intstead of being stark white like the most important elements.

I added another form of interaction where if the user hovered their cursor over the counters of the letters, it would light up yellow. I thought it would also reflect the event itself being seminars taking place in the night time, as it looked like a light in the dark when interacted with.

While I moved my large type around, I also had to keep in mind the kerning of each letter, which made for more interesting explorations.

I continued to refine the colour palette and explored with how I arranged each piece of information. I created teh gradient to create more depth in the poster as well, which I thought would add some visual interest.

I then added a few lines and circles to highlight the geometric qualities of the typography and the poster.

Final Poster

In my final design, I made sure to keep everything simple and readable. In my second html page, I made sure to reflect these qualities as well.
