WEB TYPOGRAPHIC POSTER

Inspiration

I wanted to create a minimalistic poster that would reflect my typeface, so I incorporated simple geometric motifs such as lines and circles to use within my poster.

I began by experimenting with the layout of the poster, and how each element would relate to eachother. I also considered the colour palette I was going to use in this stage, and played around with how the viewer would use the page. I used position: fixed, which would add an interesting interactive element to the poster.

I played around with how layering the elements would look, but I ultimately wanted the poster to be readable, so I only made one of the lsrger typographic elements fixed.

In establishing the hierarchy of each part, I used a combination of size and colour. The biggest text would be highest in the heirarchy, and the body text was a colour more similar to the background intstead of being stark white like the most important elements.

I added another form of interaction where if the user hovered their cursor over the counters of the letters, it would light up yellow. I thought it would also reflect the event itself being seminars taking place in the night time, as it looked like a light in the dark when interacted with.
